### Dependencies
This project uses [Vue](https://vuejs.org) and [Feather icons](https://feathericons.com)

### Setup and compilation

1. `npm install`
2. `npm run serve`

If you want to build files use, first edit `publicPath` in `vue.config.js` 
and then run `npm run build`.
